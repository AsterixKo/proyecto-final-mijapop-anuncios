Mijapop es un equipo muy reducido pero muy hábil. En 2020 se conocen en el Bootcamp de Full Stack y el azar les une para hacer esta aplicación como proyecto de fin de curso. Una de las claves para el buen funcionamiento de este proyecto ha sido el correcto uso de las skills y soft-skills de cada miembro del equipo.

El equipo de Mijapop quiere agradece que estés aquí, nos gustaría que fueses parte de nuestra comunidad, ampliando nuestro número de usuarios contigo.
